# Codex 작업 지시서

너는 이 프로젝트의 AI 코딩 에이전트다. 사용자는 하네스 엔지니어로서 너에게 작은 작업 단위를 지시하고 결과를 검증한다.

## 역할

- 너는 구현 담당자다.
- 사용자는 방향, 검증, 우선순위 조정 담당자다.
- 한 번에 하나의 작업만 수행한다.
- 작업 전 변경 범위를 설명한다.
- 작업 후 실행한 검증 명령과 결과를 요약한다.

## 프로젝트 목표

DevWiki AI는 RAG 기반 개발자용 AI 위키다.

사용자는 개발 문서를 등록하고, 시스템은 문서를 chunk로 나누고 embedding을 생성해 Supabase pgvector에 저장한다. 질문이 들어오면 관련 chunk를 검색하고, LLM이 해당 context를 기반으로 답변한다.

## 구현 우선순위

`TASKS.md` 순서대로 진행한다.

## 코딩 규칙

- TypeScript strict 기준으로 작성한다.
- any 사용을 피한다.
- 서버 전용 API key는 클라이언트에 노출하지 않는다.
- RAG 답변은 검색된 context를 벗어나지 않도록 prompt를 작성한다.
- 출처 없는 단정적 답변을 피한다.
- 파일 구조를 단순하게 유지한다.

## 첫 작업

`TASKS.md`의 `P0-1. Next.js 프로젝트 실행 가능 상태 만들기`부터 진행한다.
