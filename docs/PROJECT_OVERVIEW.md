# DevWiki AI 프로젝트 개요

## 한 줄 설명

내가 저장한 개발 지식 문서를 기반으로 AI가 출처 있는 답변을 제공하는 개인 개발자용 RAG 위키.

## 사용자 문제

개발자는 많은 문서, 발표자료, 회고, 코드 메모를 만들지만 시간이 지나면 다시 찾기 어렵다. 일반 LLM은 내 개인/회사 문맥을 모르기 때문에 구체적인 답변을 하지 못한다.

## 해결 방식

내 문서를 벡터 DB에 저장하고, 질문이 들어오면 관련 문서 조각을 검색한 뒤 LLM에게 context로 제공한다.

## 주요 사용자 시나리오

### 1. 기술 문서 저장

사용자는 React, Next.js, RAG, 테스트 코드 등 학습한 내용을 문서로 저장한다.

### 2. 질문하기

사용자는 “React Query v4로 왜 마이그레이션했지?”처럼 자신의 문서 기반 질문을 한다.

### 3. 근거 기반 답변 받기

AI는 관련 chunk를 검색하고, 해당 내용을 근거로 답변한다.

## 제품 방향

v1: 개인용 RAG 위키  
v2: GitHub 저장소/Markdown 연동  
v3: 코드베이스 RAG  
v4: 팀용 개발 지식베이스
